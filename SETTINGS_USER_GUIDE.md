# Settings Panel Navigation Guide

## Accessing Settings
Navigate to: `/settings` or click on your profile menu → Settings

## URL Structure
The settings page uses URL parameters for navigation:
- `/settings` or `/settings?tab=profile` - Profile Information
- `/settings?tab=account` - Account Settings
- `/settings?tab=payment` - Payment Information
- `/settings?tab=privacy` - Privacy & Security
- `/settings?tab=notifications` - Notification Preferences
- `/settings?tab=verification` - Verification
- `/settings?tab=delete` - Delete Account

## Common Actions

### Changing Your Password
1. Go to Account Settings
2. Enter current password
3. Enter new password (must be strong)
4. Confirm new password
5. Click "Update Password"

### Enabling 2FA
1. Go to Account Settings
2. Click "Enable 2FA"
3. Scan QR code with authenticator app
4. Enter 6-digit verification code
5. Click "Verify & Enable"

### Adding a Payment Method (Clients)
1. Go to Payment Information
2. Click "Add Card"
3. Enter card details (number, expiry, CVV, name)
4. Optionally set as default
5. Click "Add Card"

### Withdrawing Funds (Freelancers)
1. Go to Payment Information
2. Check available balance
3. Click "Withdraw"
4. Enter withdrawal amount
5. Confirm withdrawal

### Managing Notifications
1. Go to Notification Preferences
2. Toggle switches for each notification type
3. Click "Save Preferences"

### Getting Verified
1. Go to Verification tab
2. Complete each verification step:
   - Email: Click "Verify Email" and check inbox
   - Phone: Click "Verify Phone" and enter SMS code
   - Identity: Upload ID document
   - Payment: Add and verify payment method
   - Skills (freelancers): Take skill tests

### Downloading Your Data
1. Go to Privacy & Security
2. Scroll to "Data Management"
3. Click "Request Data Download"
4. Check email for download link (within 24 hours)

### Deleting Your Account
**Warning: This action is permanent!**
1. Go to Delete Account tab
2. Read warnings carefully
3. Download your data (optional)
4. Consider deactivating instead
5. Select reason for deletion
6. Click "Continue to Delete Account"
7. Check all confirmation boxes
8. Type "DELETE" in confirmation field
9. Click "Delete My Account Permanently"

## Tips & Best Practices

### Security
- Use a strong, unique password (12+ characters, mixed case, numbers, symbols)
- Enable 2FA for extra security
- Review active sessions regularly
- Check activity log for suspicious activity
- Keep email and phone verified

### Profile Completion
- Complete all profile sections to increase visibility
- Add real work samples to portfolio
- Keep skills list relevant and up-to-date
- Get verified to build trust with clients/freelancers
- Update work experience regularly

### Payment Management
- Keep bank details/payment methods up-to-date
- Set up auto-withdrawal to avoid accumulated balance
- Review transactions regularly
- Download receipts for tax purposes
- Verify payment methods promptly

### Privacy
- Review privacy settings after profile creation
- Control what information is publicly visible
- Manage connected social accounts
- Opt out of marketing emails if preferred
- Use data download for backups

### Notifications
- Keep important notifications enabled (payments, security)
- Disable marketing emails if they're too frequent
- Use push notifications for urgent updates
- Weekly digest is useful for job matching (freelancers)
- Enable proposal notifications (clients)
